Process

The following provides a high-level overview of the process from raw-dataset to great visualisations:

Step 1: Analyse dataset and make decisions — Checking dataset and determining how to leverage these data in order to visualise them to audience.

Step 2: Parse, clean and enrich the dataset — Using Excel and OpenRefine to find errors in the given dataset and fixed them.

Step 3: Build the visualisations — Combining dc.js , d3.js and other libraries to work out two basic visualisations.

Step 4: Enhance and refine visualisations — Adding other functions and adjust color elements(parameters) to improve the visualisations.
